{# =================================================================== v0.20.1 / v0.21.0 · AUTO-LOGOUT POR INACTIVIDAD El contador arranca en SESION_MIN minutos y baja solo. CUALQUIER actividad (mouse, teclado, scroll, touch) lo reinicia al maximo. Cuando llega a cero, manda a /logout-inactividad (mensaje propio). Los minutos se configuran en config.py -> SESION_TIMEOUT_MIN. v0.21.0 · NOVEDAD IMPORTANTE — el HEARTBEAT. Antes esto era SOLO cosmetico: el servidor no se enteraba de nada, y una sesion podia quedar viva para siempre si el JS no corria. Ahora la sesion TAMBIEN vence en el servidor (PERMANENT_SESSION_LIFETIME). Pero eso solo no alcanza: si vos estas LEYENDO una lista larga y solo moves el mouse, para el servidor eso es "no hizo ningun request" -> te mataria la sesion aunque estes ahi, presente. El heartbeat resuelve justo eso: cada 60 segundos, SI Y SOLO SI hubo actividad real desde el ultimo ping, le avisa al servidor "seguí viva". Si te fuiste, no hay actividad -> no hay ping -> la sesion se muere. Correcto. =================================================================== #} {% endblock %}
