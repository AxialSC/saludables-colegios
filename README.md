# Saludables · Catálogo Mayorista (AXIAL SECURITY)

Sistema Flask para la distribuidora de Juliana Maciel (Pilar).

## Stack
Flask · SQLAlchemy · Flask-Login · Bcrypt · SQLite · ReportLab

## Puesta en marcha (local)
```
pip install -r requirements.txt
flask --app run init-db
flask --app run seed-data
python run.py
```
Entrar a http://127.0.0.1:5000
